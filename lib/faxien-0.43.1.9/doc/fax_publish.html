Function Details

publish/3

publish(Repos, RawPackageDirPath::PackageDirPaths, Timeout::timeout()) -> ok | {error, Reason}

  • IsGuarded = bool()
  • Repos = [repo()]
  • ErtsVsn = string()
  • PackageDirPaths = [PackageDirPath] | PackageDirPath
  • PackageDirPath = string()

Publish a release or appliation to a repository. The PackageDirPath must be formatted in the following way <relname>-<rel version> If you are publishing a tarball it must be compressed and have the extention .epkg.

  Example:
   publish(["http://www.erlware.org/stable"], "/home/jdoe/my_proj/lib/my_app", 40000).

  Example:

publish_sinan/3

publish_sinan(Repos, CWD, Timeout) -> ok | {error, Reason}

  • Repos = [string()]

Publish from within a Sinan project. This will publish all compiled applications as well as any releases that have been created via "dist"
